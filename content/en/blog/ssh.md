Тут має бути запис про різні нашалаштування SSH.

Ось початок мого SSH конфігу:

```bash {filename=".ssh/config"}
~/.ssh/config
# ======================================================================
# Projects specific configs
# ======================================================================
Include config.d/*

# ======================================================================
# Personal hosts
# ======================================================================
Host gitlab.com
  IdentitiesOnly yes
```
